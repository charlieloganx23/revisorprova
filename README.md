# Quiz de Língua Portuguesa - 6° Ano 📚

Quiz interativo e didático para revisão de prova de Língua Portuguesa, desenvolvido para estudantes do 6° ano do Ensino Fundamental.

## 📖 Sobre o Projeto

Este quiz aborda os principais temas de Língua Portuguesa do 6° ano:

- **Linguagem Verbal** (oral e escrita)
- **Linguagem Não Verbal**
- **Texto Multimodal**
- **Interlocutores**
- **Língua** (sistema de sinais)
- **Diversidade Linguística**
- **Variação Regional**
- **Variação Histórica**
- **Variação Social**
- **Variação Situacional**
- **Norma Padrão**
- **Linguagem do Dia a Dia**
- **Preconceito Linguístico**

## ✨ Características

- ✅ **30 perguntas** completas e didáticas
- 🎯 **Sistema de pontuação** (10 a 20 pontos por pergunta)
- ⭐ **Sistema de estrelas** baseado no desempenho
- 🎨 **Interface colorida e atrativa** para estudantes
- 📊 **Barra de progresso** visual
- 🎉 **Feedback educativo** instantâneo
- 🏆 **Medalhas e confetes** para motivação
- 📱 **Design responsivo** (funciona em celular, tablet e desktop)

## 🚀 Deploy no Netlify

### Opção 1: Deploy Automático via GitHub

1. Acesse [Netlify](https://www.netlify.com/)
2. Faça login ou crie uma conta
3. Clique em "Add new site" → "Import an existing project"
4. Conecte sua conta do GitHub
5. Selecione o repositório `charlieloganx23/revisorprova`
6. As configurações serão carregadas automaticamente do arquivo `netlify.toml`
7. Clique em "Deploy site"

### Opção 2: Deploy Manual via Netlify CLI

```bash
# Instalar Netlify CLI
npm install -g netlify-cli

# Fazer login
netlify login

# Deploy
netlify deploy --prod
```

## 🎮 Como Usar

1. Abra o arquivo `quiz_portugues_6ano.html` no navegador
2. Leia cada pergunta com atenção
3. Clique na alternativa que você considera correta
4. Receba feedback imediato sobre sua resposta
5. Acumule pontos e estrelas
6. Revise os temas que você errou

## 📊 Sistema de Pontuação

- Perguntas básicas: **10 pontos**
- Perguntas intermediárias: **15 pontos**
- Perguntas avançadas: **20 pontos**

### Sistema de Estrelas

- ⭐⭐⭐ (3 estrelas): 90% ou mais de acerto
- ⭐⭐ (2 estrelas): 60% a 89% de acerto
- ⭐ (1 estrela): 30% a 59% de acerto
